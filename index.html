Scittle Kitchen

A collection of precompiled ClojureScript plugins for Scittle.
Usage:
<script src="https://timothypratley.github.io/scittle-kitchen/js/scittle.js"></script>
<script src="https://timothypratley.github.io/scittle-kitchen/js/scittle.geom.js"></script>
See the GitHub README for more info or to contribute your own plugin.

Version: v0.7.27-41manifest.edn

Scittle build that can use kitchen plugins

https://timothypratley.github.io/scittle-kitchen/js/scittle.js (dev build)
